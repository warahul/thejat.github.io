Yet another Excel reference

Date Tue 04 February 2014 Tags tools

Basic

  • Using TRIM (to trim whitespaces), PROPER( leading letter is capitalized for each word) and string concatenation (using the ampersand symbol)
    ="name: "&PROPER(TRIM(stringCell1))&" POSTCODE: "&UPPER(stringCell2)
  • Using find to get the position of a character without error handling starting at some position
    =FIND(subString,stringCell,startPositionOptional)
  • Using left and right to select a substring from a string (1st parameter) using position information (2nd parameter, a number <= len(string))
    = left(stringCell,endPosition) 
    = right(stringCell,numberOfCharachters)
  • Uppercase
    = UPPER(stringCell)
  • string length
    =len(stringCell)
  • Reference another sheet
     =sheetname!cellNumber
  • Reference another workbook
     =`[filename.xls]sheetname`!cellNumber [BAD PRACTICE! Better to not know this]
  • Counting
     =COUNTA(columnNumber:columnNumber)
     =COUNTA(A:A)
     =COUNTA(stringCell1:stringCell10)
     =COUNT(numberCell1:numberCell10)
  • sum, minimum, maximum, average,median,mode
     =sum(A1:A10)
     =min(A1:A10)
     =max(A1:A10)
     =average(A1:A10)
     =median(A1:A10)
     =mode(A1:A10)

  • Sort after removing empty/hidden columns and expanding selection.

  • Autofilter data gives selection arrows. Do not run macros on autofiltered data.

  • \(A\)1 vs A$1 vs A1

  • rounding 39.999 to 40.0

    =round(39.999,1)
  • weekday, month day, month, year encoding
    = weekday(date,1)
    = day(date)
    = month(date)
    = year(date)
    = today()
  • hour, minute on time values
   = hour(time)
   = minute(time)
   = second(time)
   = now()

Source: Based on Matthew Sands' tutorial.
